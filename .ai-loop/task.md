## Required behavior

### 1. `scripts/build_repo_map.ps1`

- Walks: `src/`, `scripts/`, `templates/`, `docs/`, `tests/`, plus repo-root `*.md`, `*.py`, `*.ps1`, `*.ini`, `*.cfg`, `*.toml`.
- Skips: `docs/archive/**`, `.ai-loop/_debug/**`, `.ai-loop/archive/**` (if exists), `.tmp/**`, `__pycache__/**`, `.git/**`, `.pytest_cache/**`, `.claude/**`, `node_modules/**`, `input/**`, `output/**`.
- For each file, extracts a **one-line purpose** by reading the leading header:
  - `.ps1`: first non-blank line inside the first `<# … #>` block matching `^\.SYNOPSIS` (take the next non-blank line); fall back to the first `#` comment line; fall back to `(no header summary)`.
  - `.py`: first line of the leading module docstring (`"""…"""` or `'''…'''`); fall back to the first `#` comment line; fall back to `(no header summary)`.
  - `.md`: first H1 (`^# …`) line stripped of `#`; fall back to first non-blank line; fall back to `(no header summary)`.
  - `.ini` / `.cfg` / `.toml` / `.json` / other: first `#`, `;`, or `//` comment line; fall back to `(no header summary)`.
- Output format (`.ai-loop/repo_map.md`):

  ```markdown
  # Repo map

  > Generated by `scripts/build_repo_map.ps1`. Do not edit by hand.
  > Regenerate after structural changes; tests pin determinism.

  ## Top-level

  - `README.md` — AI Orchestrator
  - `AGENTS.md` — Working rules for AI agents…
  - `ai_loop.py` — (no header summary)
  - …

  ## scripts/

  - `scripts/ai_loop_auto.ps1` — Codex review/fix loop with safe staging…
  - `scripts/ai_loop_task_first.ps1` — Task-first entry that runs the implementer…
  - …

  ## docs/

  - …

  ## templates/

  - …

  ## tests/

  - …
  ```

- Ordering: groups sorted alphabetically by directory; entries within each group sorted alphabetically by relative path. Determinism: running the script twice on an unchanged tree must produce byte-identical output.
- Total length: cap at **250 lines**. If exceeded, the script must `Write-Warning` and still write the file (do not silently truncate); a follow-up task will refine selection.
- Encoding: UTF-8 without BOM.
- Exit codes: `0` on success; non-zero on any I/O error.
